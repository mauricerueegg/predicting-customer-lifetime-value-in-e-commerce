# Predicting Customer Lifetime Value in E-Commerce

You are given the code for generating a dataset describing customer behaviour in an e-commerce environment, including purchase history, engagement metrics, and demographic attributes. The dataset allows you to model customer lifetime value (CLV). However, relevant information is distributed across different variables and may require aggregation or transformation. Your task is to build regression models that predict customer lifetime value and to interpret the drivers behind these predictions.

Objective
- Engineer meaningful features from behavioural data
- Train at least two regression models
- Evaluate and compare model performance
- Use SHAP values to explain model predictions

Hints
- Consider aggregating behavioural metrics into meaningful indicators
- Reflect on how past behaviour may relate to future value
- Be aware of potential skewed distributions in monetary variables
- Compare models with different assumptions (e.g. linear vs. non-linear)
- Use SHAP to understand both global and local feature effects
